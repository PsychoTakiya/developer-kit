# Glossary & Further Reading

## Glossary (Selected)

- Fiber: Internal React data structure representing component work units.
- Reconciliation: Diff algorithm producing minimal DOM changes.
- Suspense: Mechanism to coordinate async rendering with fallbacks.
- Stale Time: Duration cached data is considered fresh in TanStack Query.
- Selector: Memoized function deriving data from store state.
