# Anti-Patterns & Refactoring

## Common Issues

State sprawl in global store, prop drilling chains, oversized components, stale closure bugs, over-memoization.

## Refactoring Approaches

Introduce custom hooks, segment context, normalize state, add selectors, incremental SSR adoption.
